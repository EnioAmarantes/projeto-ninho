# Roadmap

O Projeto Ninho será desenvolvido de forma incremental, priorizando estabilidade, documentação e evolução contínua.

---

# Fase 1 — Fundação

Objetivo:

Construir toda a base técnica do projeto.

## Entregas

* Estrutura do monorepo
* Documentação
* Docker
* Pipeline CI
* Gateway
* Identity
* Banco de dados
* Dashboard inicial

---

# Fase 2 — Plataforma

Objetivo:

Disponibilizar uma plataforma funcional para gerenciamento da residência.

## Entregas

* Cadastro de ambientes
* Cadastro de dispositivos
* MQTT
* Automações
* Notificações
* Usuários
* Permissões

---

# Fase 3 — IoT

Objetivo:

Integrar dispositivos físicos.

## Entregas

* ESP32
* Sensores
* Relés
* Interruptores
* Monitoramento
* Atualizações OTA

---

# Fase 4 — Inteligência Artificial

Objetivo:

Criar o núcleo inteligente da plataforma.

## Entregas

* Ollama
* Gerenciador de modelos
* Memória
* Ferramentas
* Prompt Engine
* Persona Manager

---

# Fase 5 — Jarvis

Objetivo:

Construir um assistente realmente útil.

## Entregas

* Conversação
* Voz
* STT
* TTS
* Controle da residência
* Execução de comandos
* Contexto

---

# Fase 6 — Cyberdeck

Objetivo:

Criar interfaces físicas dedicadas.

## Entregas

* Cyberdeck infantil
* Comunicação segura
* Chamadas familiares
* Interface personalizada
* Reconhecimento de usuários

---

# Fase 7 — Ecossistema

Objetivo:

Expandir o Projeto Ninho.

## Entregas

* Plugins
* API pública
* Marketplace de integrações
* SDK
* Aplicativos de terceiros

---

# Longo prazo

* Assistente totalmente local.
* Múltiplas personas.
* Memória de longo prazo.
* Aprendizado contextual.
* Integração completa entre IA, automação e comunicação familiar.
