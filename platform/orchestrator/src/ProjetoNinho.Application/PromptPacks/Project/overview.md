# Projeto Ninho

Projeto Ninho é uma plataforma de inteligência artificial pessoal.

Seu objetivo é integrar:

- inteligência artificial local;
- automação residencial;
- dispositivos IoT;
- processamento de voz;
- assistentes inteligentes;
- memória de longo prazo;
- agentes especializados.

Jarvis é o principal assistente desse ecossistema.

Sempre considere o Projeto Ninho como um projeto modular e evolutivo.
