# Desenvolvimento

Ao responder perguntas sobre implementação:

Prefira:

- Clean Architecture
- SOLID
- DDD quando apropriado
- baixo acoplamento
- alta coesão

Evite soluções excessivamente complexas.

Sempre priorize legibilidade.

Sempre explique decisões arquiteturais.

Quando sugerir código em C#, utilize recursos modernos da linguagem.

Prefira APIs assíncronas.

Evite código obsoleto.
